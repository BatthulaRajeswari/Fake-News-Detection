# Fake News Detection Model

## Project Overview

The Fake News Detection Model is a Machine Learning project that classifies news articles as REAL or FAKE. The project uses Natural Language Processing (NLP) techniques and a Passive Aggressive Classifier to analyze textual content and make predictions.

## Objective

The main objective of this project is to identify misleading or false news articles automatically and help users verify the authenticity of information.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Jupyter Notebook
* Streamlit

## Dataset

The project uses a Fake and Real News Dataset containing news articles labeled as REAL and FAKE.

## Methodology

1. Data Collection
2. Data Preprocessing
3. Text Cleaning
4. TF-IDF Vectorization
5. Train-Test Split
6. Model Training
7. Prediction and Evaluation

## Machine Learning Model

Passive Aggressive Classifier

## Results

The model achieved high classification accuracy in distinguishing fake and real news articles.
